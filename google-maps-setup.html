‚ö†Ô∏è Important: To enable Google Maps functionality, you need to set up a Google Maps API key. Follow the steps below.

Step 1: Get Google Maps API Key

1. Go to the Google Cloud Console

2. Create a new project or select an existing one

3. Enable the following APIs:

  • Maps JavaScript API
  • Places API
  • Geolocation API

4. Create credentials (API Key)

5. Restrict the API key to your domain for security

Step 2: Add API Key to Project

In your LegalClinicFinder.js file, replace this line:

script.src = `https://maps.googleapis.com/maps/api/js?key=YOUR_GOOGLE_MAPS_API_KEY&libraries=places`;

With your actual API key:

script.src = `https://maps.googleapis.com/maps/api/js?key=YOUR_ACTUAL_API_KEY&libraries=places`;

Step 3: Test the Feature

1. Save your changes and restart the development server

2. Click on "Find Clinics" in the Legal Aid Services section

3. Allow location access when prompted

4. View nearby legal clinics on the map and in the list

Step 4: Customize Clinic Data

The current implementation uses sample data. In production, you would:

  • Connect to a real database of legal clinics
  • Use Google Places API to find actual legal services
  • Implement user reviews and ratings
  • Add filtering by specialization
